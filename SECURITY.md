# Security policy

## Supported versions

| Version | Supported |
| --- | --- |
| v0.1.x | Yes |
| Earlier versions | No |

## Reporting a vulnerability

Report vulnerabilities through the private GitHub security advisory form:

https://github.com/StringKe/goakt-gateway/security/advisories/new

Do not open a public issue, publish an exploit, or include credentials, tokens, private
keys, certificates, or customer data. Include the affected version, a minimal reproducer,
impact, and a secure contact channel.

The maintainer will acknowledge valid reports, assess the affected versions, and coordinate
the disclosure and release through the private advisory.
